KeePass 2.0.0 ML (Multi-Language)
Encrypted Password Safe.

Tested on Windows 7, 8.1 + 10

Slient Install: msiexec /i "Keepass_2.0.0_ML.msi" /qn (absolute silent) or /qb! (with progress-dialog).

All MSI-versions from Keepass are found and automatically removed. Only one version can be installed.

Features in this version:
- Multi-Language enu, deu, fra, spa.
- Autoupdate disabled.
- No desktop Shortcut.
